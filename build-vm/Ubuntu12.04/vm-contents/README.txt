README
*********

1. Go to Tools->Extension Manager and then click on "Add".
2. Add both the extensions one-by-one from "Add-on-Extensions" folder on the Desktop and click on "Close".
3. Make a worksheet of your own by adding data on Sheet1 and name it.
4. Delete the other worksheets Sheet2 and Sheet3.
5. Click the inference button on the toolbar which looks like "A->B".
6. Your Class model and instance spreadsheet will be created automatically.

Note:
Add-on-Extensions folder on the desktop contains the MDSheet extensions (for inference feature).
