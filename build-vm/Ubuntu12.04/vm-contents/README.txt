README
*********

1. Click on the Libreoffice -calc icon on the left-side of the Desktop.
2. Go to Tools->Extension Manager and then click on "Add".
3. Add both the extensions one-by-one from "Add-on-Extensions" folder on the Desktop and click on "Close".
4. Make a worksheet of your own by adding data on Sheet1.
5. Delete the other worksheets Sheet2 and Sheet3.
6. Click the inference button on the toolbar which looks like "A->B".
7. Your Class model and instance spreadsheet will be created automatically.

Note:
Add-on-Extensions folder on the desktop contains the MDSheet extensions (for inference feature).
